You will need to include the Convalida JAR in your application's runtime. In order to activate code generation you will need to include the compiler JAR in your build at compile time.

The source code to the Convalida, its samples, and this website is available on GitHub.

Maven

Step 1: Add the Jitpack repository

<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>

Step 2: Add the dependencies

<dependency>
  <groupId>com.github.WellingtonCosta.convalida</groupId>
  <artifactId>convalida</artifactId>
  <version>(insert latest version)</version>
</dependency>
<dependency>
  <groupId>com.github.WellingtonCosta.convalida</groupId>
  <artifactId>convalida-compiler</artifactId>
  <version>(insert latest version)</version>
  <optional>true</optional>
</dependency>

Gradle

Step 1: Add the Jitpack repository

allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}

Step 2: Add the dependencies

dependencies {
  implementation 'com.github.WellingtonCosta.convalida:convalida:(insert latest version)'
  annotationProcesso 'com.github.WellingtonCosta.convalida:convalida-compiler:(insert latest version)'
}
